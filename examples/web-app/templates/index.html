Auth47 Web App Example

Demonstrating Bitcoin-based authentication using the Auth47 protocol

Secure Authentication

Auth47 uses Bitcoin payment codes and cryptographic signatures to provide secure, decentralized authentication without passwords.

QR Code Login

Simply scan a QR code with your Bitcoin wallet to authenticate. No typing passwords or remembering credentials.

Try Authentication

How It Works

1️⃣
Generate Challenge

Server creates a unique Auth47 challenge with a nonce and expiry time.

2️⃣
Sign with Wallet

User scans QR code and signs the challenge with their Bitcoin wallet.

3️⃣
Verify & Access

Server verifies the signature and grants access to protected resources.

Features Demonstrated

  • Challenge Generation: Creates secure Auth47 URIs with nonces and expiry
  • QR Code Display: Mobile-friendly authentication using QR codes
  • Proof Verification: Validates Bitcoin signatures and payment codes
  • Session Management: Maintains authentication state securely
  • Protected Routes: Demonstrates access control for authenticated users
